Wallet App

A simple and secure Wallet application to manage your funds, track transactions, and make payments easily. Built with Python and PyQt6.

Features

User Authentication: Sign up and sign in securely.

Manage Funds: Add, remove, and track your wallet balance.

Transaction History: Keep a record of all deposits and withdrawals.

Multi-Currency Support: Works with Bitcoin, Ethereum, and Tether.

Email Verification: Ensures secure account registration.

Easy-to-Use Interface: Built with PyQt6 for a smooth user experience.
